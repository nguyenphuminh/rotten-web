RottenJS





What ?

RottenJS is a Javascript library that helps make DOM manipulations and UI development much easier! It is good for both small and large projects.



Why ?

RottenJS is extremely fast and light-weight (5kb). It can definitely scales up to medium to large scale projects. RottenJS is very easy to learn with the friendly documentations and setup without any toolings. Furthermore, it is also fully open-source on Github.
